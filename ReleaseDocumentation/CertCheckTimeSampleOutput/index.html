After installation of the compute element, and execution the fetch-crl cron entry, you can check the remaining lifetime of the installed CAs as shown below. See full man page of the command for additional options. 

 > cert-check-time -cR
Result: shows the remaining lifetime in days, the DN of the CA, and the full path to the CA certificate file. With these options, it also flags an CAs that do not have a CRL. If the CA publishes a crl it should be indicated in the CA-hash.info file so you may want to check that for those CAs flagged without a CRL.
