RSV - WLCG SAM data exchange

NOTE: Arvind needs to edit this RSN..

One of the less talked about objectives of RSV? is the RSV-WLCG SAM data exchange. The OSG GOC will put an infrastructure in place to transfer resource and service availablity metrics from the central RSV database to the WLCG SAM system. This data transfer is expected to be reasonably streamlined and straight forward since both parties will be conforming to the the current version (0.91) of the WLCG grid monitoring probe specification.

Once the infrastructure is in place, and completely automated (EOT: Dec 2007 for ITB CE data; Jan 2008 for OSG production CE data), WLCG SAM will not probe individual OSG resources. Instead it will pick up availability data from the RSV database.

More information on this topic will be made available in the near future. If you have any questions. please contact the RSV development team.
