List of Gratia Collectors Operated by FermiGrid

OSG services

  • All services run on their own VM;
  • All services listen on ports 80 (HTTP) and 443 (HTTPS);
  • All services run out of /data/tomcat-gratia;
  • All init.d service scripts are named tomcat-gratia;
  • Legacy connections are forwarded to the correct collector by gr8x5.fnal.gov which manages legacy aliases as service IPs.

gratia-osg-prod.opensciencegrid.org:80 gr5x0.fnal.gov 8880, 8845 osg-prod gratia Status Reports For batch records (probe types pbs, sge, condor, lsf).
gratia-osg-daily.opensciencegrid.org:80 gr5x1.fnal.gov 8884, 8860 osg-daily gratia_osg_daily Status Reports For daily summary info only (if you're not sure, you don't need this).
gratia-osg-transfer.opensciencegrid.org:80 gr5x2.fnal.gov 8886, 8840 osg-transfer gratia_osg_transfer Status Reports Storage / transfer (probe types dcache-storgage:, dcache-transfer:, gridftp-transfer).
gratia-osg-itb.opensciencegrid.org:80 gr5x3.fnal.gov 8881, 8850 osg-itb gratia_itb Status Reports ITB collector (all probe types).
gratia-osg-integration.opensciencegrid.org:80 gr5x4.fnal.gov 8885, 8870 osg-integration gratia_osg_integration Status Reports Selected "permanent ITB" probes replicated from gratia-osg-itb. Do not configure probes or non FNAL collectors to send here.

Fermi-local services

  • All services run on their own VM;
  • All services listen on ports 80 (HTTP) and 443 (HTTPS);
  • All services run out of /data/tomcat-gratia;
  • All init.d service scripts are named tomcat-gratia;
  • Legacy connections are forwarded to the correct collector by gr5x5.fnal.gov which manages the gratia-fermi.fnal.gov service IP.
