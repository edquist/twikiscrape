IPv6 Testing Log: VOMS

This testing log is part of OSG IPv6 tests. This page captures the specific IPv6 compatibility testing process used for one software component, VOMS. The more general IPv6 template for testing is here.

Initial Setup

This test used two hosts:

Host OS OSG Release
cabinet-10-10-4.t2.ucsd.edu CentOS? release 5.9 (Final) 3.2
ipv6vm006.fnal.gov Scientific Linux Fermi release 6.6(Ramsey) 3.2

Installing software

Next, OSG software was installed on each system. The UCSD host was the client and the FNAL host was the server. And VOMS software was installed following the instructions here VOMSInstallation
