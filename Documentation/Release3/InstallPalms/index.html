2 Introduction

This document describes the installation and use of the OSG Programs, Applications Libraries Management and Setup (PALMS). PALMS is a software and a set of procedures the OSG will develop for others to use and also a value-added service the OSG may offer to smaller VOs together with OASIS to assist with the distribution of VO application software. This service will be provided as complement to OASIS to facilitate the tasks of VO software manages in OSG.
