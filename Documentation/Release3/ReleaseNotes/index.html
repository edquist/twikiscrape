What's New in Release 3

Release 3 introduces use of RPMs for installation of most of the software packages in OSG. This is a major change in methodology that affects nearly every aspect of installing the release.

Requirements

  • OS: Scientific Linux 6 or 7, CentOS 6 or 7, or Red Hat Enterprise Linux 6 or 7
  • Make sure you don't have Globus pre-installed from EPEL.

NOTE: Release 3.2 not supported anymore

Since August 9, 2016, release series 3.2.x are not supported anymore.
