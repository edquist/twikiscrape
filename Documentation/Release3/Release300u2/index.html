OSG Software release 3.0.0u2

Release Date: 31-Oct-2011

Summary of Changes

  • The VOMS server (including VOMS Admin) is now a general release.
  • Java has been updated to a new version (6.0.0u29) to address various security problems.
  • The OSG client, OSG Worker Node Client, and glexec packages are now in limited production release. That is, they are released to production and will be supported, but we will only encourage early adopters to use them.

All of the documentation can be found in the "Release3" area of the Twiki. For this release, you may want to take particular note of:

  • Installing VOMS
  • Upgrading from the Pacman-based VOMS

The early adopters among you may wish to read:

  • Installing the worker node client
  • Installing glexec
  • Installing the client
