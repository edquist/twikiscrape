    • Document rating/collection of ratings (PeerPlugin?)

  • Comment: current use of %INCLUDE% needs to be reviewed. For example, some docs are nothing but included documents that are used no where else.
  • Having the new TWiki for testing our technologies, plugins, process, use of document flags, etc. is a great idea -> We can move proven items back into our current TWiki
