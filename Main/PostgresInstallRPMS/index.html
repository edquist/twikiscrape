For Scientific Linux 3, see the links in the dCache Book on the page Installing a PostgreSQL Server

If you need to install postgres for Scientific Linux 4, use the rpms for Red Hat Enterprise Linux 4, from http://www.postgresql.org/ftp/binary/v8.1.0/linux/rpms/redhat/rhel-es-4/. You can optionally install the perl DBI module perl-DBD-Pg-1.31-5.i386.rpm, in which case you will need to get http://developer.postgresql.org/~devrim/compat-postgresql-libs-3-3PGDG.i686.rpm.
