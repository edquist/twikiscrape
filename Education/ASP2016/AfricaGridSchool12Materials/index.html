2012 Africa Grid School Materials

Note to lecturers preparing material - here is the OSG User School page that this page was derived from. All material is available there.

Monday

Sometime today (you can choose when), please get a certificate. You don't need it today, but you'll need it Tuesday morning.

  • Get a certificate

Monday Morning 1

  • Use of Grid Computing in HEP (Slides needed)
  • Condor Lecture #1 (PowerPoint, PDF)
  • Introduction
  • Our Condor installation
  • Your first Condor job
  • A few tips and tricks

Monday Morning 2

  • Condor Lecture #2 (PowerPoint, PDF)
  • A standard universe job
  • A scripting job
  • Working with data in files
  • An R job

Monday Afternoon 1

  • Condor Lecture #3 (PowerPoint, PDF)
  • A brief detour through the Mandlebrot set
  • Coordinating set of jobs: A simple DAG
  • A more complex DAG
  • Handling jobs that fail with DAGMan
  • Simpler DAGs with variable substitutions

Monday Afternoon 2

  • Condor Lecture #4 (PowerPoint, PDF)
  • Running a real science job: BLAST
  • Condor challenges (If and only if you have time)

Tuesday

Tuesday Morning: Distributed HTC & Overlay systems - Part 1

  • Recap of Day one and Intro to the Open Science Grid (ODP)
  • Lecture - Introduction to Distributed HTC and overlay systems (ODP, PDF)
  • Our glideinWMS setup
  • Exercise - Using the glideinWMS - Part 1
  • Exercise - Using the glideinWMS - Part 2
  • Exercise - Using glideinWMS without a dedicated node - Advanced, only if you have spare time

Tuesday Morning: Security, Distributed HTC & Overlay systems - Part 2

  • Interactive session - Introduction to security
  • Lecture - Security in OSG (ODP, PDF)
  • Exercises - Using OSG Security tools
  • Lecture - Introduction to the Grid and the glideinWMS architecture (ODP, PDF) Time Permitting

Tuesday Afternoon: Storage

  • Storage Presentation (PPT, PDF)
  • Exercises - Determining Data Requirements
  • Exercises - Hands-on with SRM and GridFTP
  • Exercises - Hands-on with HTTP and Remote I/O (on your own)

Wednesday

Wednesday Morning:

  • Lecture: Introduction to Analysis Example tutorial slides pptx,PDF
  • AnalysisExample2012

Wednesday Afternoon: Surveys and Wrap-UP

  • Student Surveys
  • Grid Workshop Wrap-Up
