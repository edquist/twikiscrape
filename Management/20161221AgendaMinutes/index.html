Susan Sons - Security (Updates since October 26)

Vulnerabilities/Incidents

A Linux kernel vulnerability known as DirtyCOW?, which allows privilege escalation was reported by the end of October, 2016. OSG received the report from EGI: https://wiki.egi.eu/wiki/SVG:Advisory-SVG-CVE-2016-5195 It consists of a race condition found in the way the Linux kernel's memory subsystem handled the copy-on-write (COW) breakage of private read-only memory mappings. By exploiting this vulnerability, an unprivileged local user could gain access to read-only memory mappings and increase her privileges on the system. OSG sent multiple announcements to the community with an initial assessment of the vulnerability and later with updated links for security patches.
