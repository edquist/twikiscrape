Adding a New Document to DocDB

Use this procedure only when adding a completely new document to DocDB? that has an attachment, for any updates or documents referring to links please see below.

  • Open DocDB
  • If you are unsure of where to place the document you can instead click on Guidelines on Which Topics to Use
  • Select Certificate Version of OSG DocDB?
  • Select Create or change documents or other information
  • Scroll down to the orange Customized Insert/Modify box
  • Select New Document
  • If this is a Policy, be sure to select the appropriate Keyword as specified here.
  • Submit all relevant information.
  • Be sure to enter the document in the correct topic (revisit the Topic Guidelines if you are unsure) and the correct keyword if it is applicable.

Updating a Document in DocDB

  • Open DocDB
  • If you are unsure of where to place the document you can instead click on Guidelines on Which Topics to Use
  • Select Certificate Version of OSG DocDB?
  • Select Create or change documents or other information
  • Scroll down to the orange Customized Insert/Modify box
Here you have two options:
  1. Select Update DB information if just changing location or other document database information. When selecting Update DB information be sure to select override submit date in the bottom of the box, and re-enter the original date.
  2. Select Update Existing Document if uploading a new version of a document, which must include a new attachment. When updating version of attached document DO NOT override submit date. It should be current for the document version. This option creates an actual new version of the document, EXAMPLE: "OSG DocDB? Document 48-v1". All other changes are available in Update DB Information.

Creating a document with a link to an exterior page

  • Open DocDB
  • If you are unsure of where to place the document you can instead click on Guidelines on Which Topics to Use
  • Select Certificate Version of OSG DocDB?
  • Select Create or change documents or other information
  • Scroll down to the orange Customized Insert/Modify box
  • Select Reserve DB #
  • Complete form, with link to exterior located in the abstract
