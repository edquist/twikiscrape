First agenda item

BeStMan2 release

Discussion

BeStMan2 2.0.14 will be released by the end of this week. This will be the last version of BeStMan2 packaged in pacman. Alex and his team are working on modification of configuration script so it is compatible with native packaging standards. There are still some issues that need to be worked out. We have discussed how to preserve consistency and avoid unnecessary redundancy in bestman2.rc and /etc/sysconfig/bestman2 files.
