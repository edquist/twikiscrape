Agenda items

1. D0 issues.
Abhishek reported that D0 is making a good progress. UCSD has made modification on CE node to limit the number of simultaneously submitted D0 jobs and it looks like D0 is able to use this site productively. There is no feedback from MIT and we need to discuss with Purdue the status of Opportunistic Storage. Abhishek has promised to contact both sites.

2. SBGrid.
BeStMan RSV probes issue has been resolved. Goc ticket has been updated.

3. Gratia probes for BeStMan:
Brian will re-use code written for GIP/BeStMan to create gratia storage probe for BeStMan
Andrew will use existing gratia-transfer-probe for dcache and gridftp log parser provided by Dan Gunter to
create gratia transfer probe for BeStMan

4. Brian will implement time stamp in gratia dcahe probes to decrease the number of records sent to gratia on startup.
Tanya has to submit a ticket to GOC for reminder

5. All BeStMan-gateway/xrootd software is in VDT, all but xrootd could be configured and start/stop by using vdt scripts. Tanya asked for feed back on Installation Guide from BeStMan and Xrootd teams

6. Neha asked for feedback for documentation update. Documentation can be found at:

https://twiki.grid.iu.edu/twiki/bin/view/Documentation/WebHome

End Users: OSG Storage Element

Abhishek pointed out on some obsolete links. Neha will re-design them.

7. Ted and Tanya have provided a quick summary of dcache workshop at Fermilab (details at http://indico.fnal.gov/conferenceTimeTable.py?confId=2085)
