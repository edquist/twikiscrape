2. Discussed what kind of performance tests should be done for bestMan/xrootd and
is it possible to compare performance of dcache & bestMan/xrootd .

Alex and Ted didn't think that it would be a very meaningful comparison. Tanya has disagreed. She thought that it would be possible to compare performance of two storage installations for different workflow patterns. She will need to understand more about xrootd and will come up with examples for potential performance tests. Alex has promised to send the recent performance test results for xrootd.

3. Discussed vdt installation of BeStMan/xrootd (will be "unsupported" software - without any configuration scripts for xrootd installation). Tanya mentioned that the xrootd.rpm refference in LBNL BeStMan webpage looks like obsolete rpm. Alex said that xrootd should be download from SLAC and all the questions should be send to Wei Yang. Tanya has got two nodes from CMS to install BeStMan/xrootd. She has installed only BeStMan and FUSE for now. Need help with xrootd configuration (hopefully Andrew will be able to help when he comes from vacation)
