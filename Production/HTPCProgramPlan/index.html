OSG HTPC Workplan

This project is owned by OSG Production as a collaboration with HTPC Last update: 28-Feb-2012, Gore/Fraser

Goal:

Make HTPC a Production quality service on the OSG in 2012. (Success depends largely on adoption by CMS and Atlas sites.)

Objectives:

  1. Stabilize existing HTPC capability at OU, Clemson, and Purdue. (Feb)
  2. Test HTPC Accounting for PBS, LSF, and Condor (Mar)
  3. Help CMS try out and use HTPC – Targeted test site is Purdue (Mar - May)
    • Also start working with Atlas as appropriate
  4. Introduce an HTPC RSV probe into the OSG to make sure sites that are supporting HTPC are functioning. (Mar)
  5. Support the deployment of Condor’s improved Partitionable Slot Technology (Apr)
  6. Integrate HTPC documentation and best practices into the OSG CE install guide (June)
  7. Broaden CMS usage of HTPC as the updated version of Condor is moved to production (July – Sep, depends on CMS timetable)
  8. Engage interested Atlas sites to use HTPC (July – Sep) depends on Atlas timetable)
  9. Make HTPC sites discoverable via BDII ?? (Sep)
