OSG Software Team

Discussions

VDT website. We (in Madison) are making some changes to ACLs in the vdt.cs.wisc.edu web area. They should not affect any use cases (that we care about), but if you notice any new issues, please let us know right away. Mostly, no one uses the old VDT website anyway.

GRAM/PBS failures. No progress last week. TimC will talk to Jeff about whether Hyak is ready for ITB pilots, and will assign someone from the software team to start thinking about debugging. Also, we will need to reproduce the issue.

Repo consolidation. There is a new little project to consolidate all of our public software repositories to repo.grid.iu.edu, thereby allowing the GOC to decommission software.grid.iu.edu. At the moment, it seems that this change will affect only certain CA certificate use cases and any software that depends on the old host.

EL 7 support. The three main runtime failures in automated testing are with Gratia, MyProxy, and PBS, and most of the failures are due to service start-up issues and the MySQL → MariaDB transition; Carl is fixing the tests and patching Gratia (at least) and has some confidence that tests will be cleaner by release. Mat has built GUMS and BeStMan, but is still working on GUMS runtime issues with low confidence that anything will be ready in time for the November release.

EL 7 on FermiCloud. Wrote Steve Timm, “We are dependent on the Scientific Linux team blessing Scientific Linux 7 for official use at Fermilab and official confirmation that it conforms to the Fermilab baseline.” In the meantime, contact Mat for help on how to do EL 7 testing. Also, John offered (again) the use of BNL facilities to do EL 7 testing on VMs. John and Mat will write an internal TWiki document with the different methodologies for EL 7 testing and share the link with the Software team.

GPU support in GlideinWMS. Wrote Marco, “I added a feature in GlideinWMS to tell glideins to auto discover GPUs using condor_gpu_discovery and advertise them to user jobs but I have no node with GPUs to test it. Is it there any cluster with GPUs that I could access?” Yes, Nebraska! Derek will reply to Marco.
