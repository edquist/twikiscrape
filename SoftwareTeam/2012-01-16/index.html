OSG Software Team Meeting for 16-Jan-2011

January 16th was Martin Luther King Jr. Day, so the meeting was not held. Instead we collected status reports by email, which are below.

Update to OSG 1.2.26

We need to do a security release for the Pacman-based OSG software. It will fix a serious security flaw in Tomcat 5.5 and it will fix a problem with the Gratia PBS probe.
