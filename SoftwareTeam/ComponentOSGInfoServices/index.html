OSG Software Component: OSG Info Services

Purpose

OSG Info Services takes data that GIP generates and uploads it to the BDII and RESS servers. See the OSG Info Services Overview for more info about these components. OSG Info Services also provides a replacement for CEMon in OSG 3.2.

Usage

/usr/sbin/osg-info-services can be run directly, otherwise it is run by cron at 10 minute intervals (+/- up to 5 minutes from a random sleep). The init script enables/disables the service being run by cron.

Source

The source for osg-info-services currently lives in the UW SVN under the software area.

Packaging

Repository: OSG.

History

Brian Bockelman wrote the original version of the software, then turned it over to the OSG Software team in 2013.
