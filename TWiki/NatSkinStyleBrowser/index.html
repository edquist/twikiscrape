Style Browser

Explore the style variations of the NatSkin by changing the base style and a variation of it. In general style variations are used to change the header art but also adapt colors to match the graphics. Various elements of the style can be relocated using style switches or even be switched off. Use the "reset" button whenever you want to revert your selection and come back to the site's default settings. Note that each web may have its own defaults build in. If you select different style options they have precedence over those.

Register here? to make your settings persistent. See the NatSkinPlugin? documentation on how to use style settings in your personal TWiki account.
