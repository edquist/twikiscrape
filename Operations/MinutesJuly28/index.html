Operations Agenda

Followup items from the previous meeting

  • OSG 1.0.0 installations - Several resources plan to upgrade this week.
  • DNS Cache Poisoning
    • With DNS poisoning, an attacker can redirect users to another DNS servers. This puts visitors in jeopardy of downloading trojan horses. Steve Timm recommends this site with a "check DNS" feature
  • CEmon resources not reporting after restart - Some users are still reporting to the old DNS.

Hot Topics for the week

  • OIM Web Updates
    • Public views of registered resources, VOs, SCs, etc. Someone without a certificate can access a limited amount of information, but can't change anything
    • Additional miscellaneous contacts can be added now - allows more people to edit a resource ... if they are added by an existing contact.
    • Users with no registered DNs can access public information on the secure URL.
