RSV WLCG SAM Equivalences

This page expands on availability (a.k.a critical) metrics for various OSG services in the WLCG Sam/GridView availability calculations. The SAM team maintains a similar page for their reference. This list is used by the SAM/GridView system, the RSV reporting system as well as the central RSV status interface.

Critical Tests for OSG Resources

See if a certain test is marked as an availability metric for that service in the tables below.
