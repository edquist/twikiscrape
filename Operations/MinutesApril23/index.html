Followup items from the previous meeting

A couple more sites (OSG_LIGO_PSU & Vanderbilt) have announced they will support the Engage VO. If you can support them, please do so and notify the GOC so that the information can be recorded in our ticket.

Maintenance Tool - Wayne asked for a text-based tool from which to grab data with site tools. GOC will look into this. VORS provides a text-based interface, but maintenance is set manually. The Maintenance Tool currently provides an XML format as well.

Hot Topics for the week

1) VO spotlight - CompBioGrid

They have 13 sites that they can access with jobmanager Fork. 300 batch slots for peak is their goal. They will send the GOC a list of sites where they can run successfully currently. It will be two to three months time before they get up to a production level and start dealing with issues.

2) DZero Production Run Update

7.7 million events processed in one day over the weekend, their best yet! 300 of the 500 million events have been run. They consider this run to be going very well.
