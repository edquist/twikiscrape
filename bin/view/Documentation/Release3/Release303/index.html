OSG Software release 3.0.3

Release Date: 29-Nov-2011
Release Date: 22-Nov-2011

Summary of Changes

29-Nov-2011:

  • This is not a new release (no changes occurred in the software repository), but we are declaring that this release is the general release of the Client, Worker Node Client & glexec.

22-Nov-2011:

  • Xrootd is now in limited production release.

All of the documentation can be found in the "Release3" area of the Twiki. For this release, you may want to take particular note of:

  • Installing Xrootd

We updated the following new packages to the production OSG yum repository. Note that in some cases, there are multiple RPMs for each 'package'.
