OSG Software release 3.0.6

Release Date: 31-Jan-2011

Summary of Changes

  • Updated to Globus 5.2.0. (Previously we had a pre-release of Globus 5.2.0.)
  • Many improvements to osg-configure. Of particular note:
    • SOFTWARE-315: osg-configure now assists with configuration authorization for your CE.
    • SOFTWARE-367: You can now configure firewall information for Globus.
    • SOFTWARE-351: You can now turn off the the Globus GRAM SEG
    • Many bug fixes and small improvements.
  • Many improvements to RSV. Of particular note:
    • SOFTWARE-450: Allow RSV to work with Condor that was not installed via RPM
    • SOFTWARE-253: Allow parameters to probes (needed for storage probes)
    • Many bug fixes and small improvements.
  • Upgrade to Condor 7.6.6 SOFTWARE-446
  • SOFTWARE-452: The osg-system-profiler was improved and collects more information
  • SOFTWARE-363: There are now scripts that clean up old files created by Globus. This is a port of the VDT-Cleanup-Scripts package from OSG 1.2.x
  • A new version of Bestman2
  • Update to latest glideinwms VO Frontend
  • A Gratia probe update (includes rewritten Condor probe)

All of the documentation can be found in the "Release3" area of the Twiki.

Updating to the new release

If you have an existing installation, you can update to this new release with: 

[root@client ~]$ yum update

NOTE
Please be aware that running yum update may also update other RPMs. You can exclude packages from being updated using the --exclude=[package-name or regex] option for the yum command.
