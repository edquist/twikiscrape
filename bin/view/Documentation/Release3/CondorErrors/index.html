Hold codes organized by the reason for the hold.

  • Unspecified
    • Hold code 0
  • Globus error
    • Hold code 2
  • Proxy
    • Hold code 4
  • Placed on hold by user or user's job
    • Hold code 1
    • Hold code 3
    • Hold code 5
  • File operations
    • Hold code 6
    • Hold code 7
    • Hold code 8
    • Hold code 14
  • Input/Output
    • Hold code 9
    • Hold code 10
    • Hold code 11
    • Hold code 12
    • Hold code 13

When a job goes on hold, an easy way to see the full reason, code, and sub code is to use condor_q -l jobid. For documentation on hold codes for your particular version of Condor, see the Condor Manual.
