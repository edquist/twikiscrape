OSG Software release 3.0.7

Release Date: 14-Feb-2012

WARNING!
We have found a critical bug in the version of the globus-gram-job-manager package included in this release. Any users that have installed globus-gram-job-manager-13.14-1.2.osg.el5 should run yum downgrade globus-gram-job-manager. We plan to deliver a new version with a bugfix when it is ready.

Summary of Changes

  • The CA certificate distribution has been updated to include the latest IGTF 1.44 CA certificates.
  • Bestman2, Condor, Globus-GRAM-Job-Manager and RSV have been updated for minor bug fixes.
  • SOFTWARE-443: Changed globus-gatekeeper to use OSG firewall information.

All of the documentation can be found in the "Release3" area of the Twiki.

Updating to the new release

If you have an existing installation, you can update to this new release with: 

[root@client ~]$ yum update

NOTE
Please be aware that running yum update may also update other RPMs. You can exclude packages from being updated using the --exclude=[package-name or regex] option for the yum command.

Detailed Changes
