Globus Troubleshooting

Common Globus Error Codes:

3: an I/O operation failed

5: GRAM Job failed because the executable does not exist

7: GRAM Job submission failed because an authentication operation failed

12: GRAM Job submission failed because the connection to the server failed (check host and port)

17: GRAM Job failed because the job failed when the job manager attempted to run it

22. GRAM Job submission failed because the job manager failed to create an internal script argument file

39: GRAM Job submission failed because the provided RSL string includes variables that could not be identified

43: the job manager failed to stage the executable

44: GRAM Job failed because the job manager failed to stage the stdin file

47: the gatekeeper failed to run the job manager

73: GRAM Job submission failed because the job manager failed to open stdout

74: GRAM Job submission failed because the job manager failed to open stderr

93: GRAM Job submission failed because the gatekeeper failed to find the requested service

122: could not read the job state file

129: the standard output/error size is different

131: the user proxy expired (job is still running)

135: GRAM Job failed because the job manager could not stage in a file

136: the scratch directory could not be created

155: GRAM Job failed because the job manager could not stage out a file
