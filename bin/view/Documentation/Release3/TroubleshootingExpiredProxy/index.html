Expired Proxy

Syntax:

ERROR: proxy has expired

Known causes:

Your proxy has expired

Recommendations:

  • Obtain a new proxy using: 
$> grid-proxy-init
or 
$> voms-proxy-init

Submit the job to the grid as the same user whose certificate was authenticated with grid-proxy-init.

i.e. *do not do*: 

$> grid-proxy-init
$> su
$> globus-job-run <site> <job>

See Also

  • Globus Authentication failures
