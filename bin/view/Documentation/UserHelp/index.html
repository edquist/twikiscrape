About This Document

This document is identifies ways a scientist/user of OSG can get help.

Documentation

Various documents may be of help.
  • If you are new to OSG we recommend you to check this introduction.
  • For basic information about running jobs check UsingTheGrid.
  • For troubleshooting, check Troubleshooting Guide
  • For a complete list of documentation for users go to User Navigation.

For Help from a Person

  • You can get help by sending email to OSG User Support.
  • Each VO typically provides a VO Support Center and help infrastructure.

Help with Specific Issues

  • If you are already using OSG and are having problems, you may open a ticket about the resource or the software that is causing problems.
  • You can also send email to the OSG Grid Operation Center (which also accept general inquiries not intended for tickets).
  • The site coordination group provides regular office hours support available weekly on a regular schedule using a Campfire Web chat. The web chat is mainly targeting Site support but any OSG user is welcome to connect to receive support. The current schedule for the chats and guest login information can be found in the Chat Calendar?.

References

  • For a more complete (and more complex) document on ways that system administrators and others can get help from OSG and VOs please check Help Procedure.
