First agenda item

BeStMan2 client in ITB

Discussion

The client commands will be included in the next ITB release of OSG client (in a couple of weeks). The srm-lbnl-client commands will still be used by default, users will need to execute the additional setup command to have access to the BeStMan2 client. If this works ok we will hope to make BeStman2 client as default but still allow users to switch back to srm-lbnl-client.
