Storage Storage Activity Meeting Minutes 2008 May 28

Attending

Attending: Ted Hesselroth, Neha Sharma, Brian Bockelman, Alex Sim

Open tickets were discussed. Neha's reports follows the minutes. Dmitry Litvintsev said he would help Neha set up kerberos testing for ticket 4100. Ted will follow up with Tigran Mkrtchyan on ticket 4472. Ted will work with Xin Zhao on the replica manager problem.

Gridworks pools are currently being borrowed by the Fermi dCache developers. Alex has begun testing lcg-utils against Bestman. He is using the CERN build. Ted mentioned that there is a VDT pacman build for them. Ted will send Alex the script he is using for testing lcg-utils.

Ted has added mappings to gPlazma to map DN->UID and Group/Role->GID. He is also working with Jay Packard on changes to the privilege java code for the XACML interoperability project.

Brian reported that ITB testing of the dCache GIP probes is nearly complete and he expects a final version out today. Brian will send Alex information on writing GIPs for Bestman.

UNL has some new tools which Brian will talk to Abhishek Rana about adding to the OSG Storgage toolkit.

Alex Sim reported that Bestman version 2.0.10 will be in the next VDT release.

Alex made a request for examples of processing on CEs to be used in a demo at SuperComputing?.
