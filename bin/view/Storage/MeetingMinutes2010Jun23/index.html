Fourth agenda item

BeStMan2

Discussion

BeStMan2 is released to VDT, It is included in vdt-rpm bestman/hdfs distribution as well as in vdt-pacman test cache. Alex provided the following information about BeStMan2:
  1. Functionality didn’t change
  2. It is using new web service container jetty, The packaging is completely different.
  3. The configuration options are almost the same (http-public-port is not needed an more)
  4. Decreased dependency of globus libararies
  5. Main challenge is to understand how to tune up jetty and increase scalability. There is not a lot of documentation provided.
