Storage Meeting Minutes 2009 Jan 28

Attending

  • Ted Hesselroth
  • Alex Sim
  • Abhishek Rana - sent report
  • Tanya Levshina

First agenda item

Transfer gratia probes

Discussion

dCache gratia probes upgrade is ready and has been installed already in couple of places. The rpms and instruction for vdt-dCache installation could be found here and for vanilla dCache installation here.

The probes have been built with postgress 8.1.9 lib that corresponds to the library used in vdt-dCache official release. For sites that are using postgress 8.3.5 we will need either to build rpms or provide them with instructions how to do it from the source. Brian has promised to clarify with site admins the best way to do it.

Gridftp transfer probe is ready to be built, we are waiting for Chris to release rpms.

Second Agenda Item

D0 problems with storage.

Discussion

D0 has reported two problems last week. The problem at Purdue looks to be partially solved by increasing the timeout and decreasing the number of the simultaneous write queues for a single pool allocated for D0.

There were no progress in addressing problem at MWT2_IU_SE. The problem seems to be also due to single pool overload. Ted & Sarah are looking into that.

Third agenda item

dcache 1.9.2 status

Discussion

It is highly recommended to move to postgress 8.3.5. This require a new pnfs rpm that is not released yet. Vladimir has promised to build non-official rpm and update instructions how to do pnfs database upgrade.

There is a reported problem that user failed to allocate space when FQAN doesn't have role. Ted is looking into it.

Fourth agenda item

BeStMan release

Discussion

Alex will release a new version of BeStMan that will include a fix for "localPathListAllowed" option in a week. We also should ask Scot to add this option for vdt configure-bestman. This was requested by US ATLAS.

Fifth agenda item

DCAU options in srm clients and dcache

Discussion

dCache doesn't support DCAU option so lcg-utils and other srm clients are using nodcau option to access dcache & globus gridftp. The discussion is going on if this could be a potential problem. See GOC ticket #6250.

Sixth agenda item

AHM

Discussion

Alex has agreed to do two presentations at AHM Foundation workshop lead by Ian Stokes-Rees. He is preparing slides for bestman (10 slides=~10 minutes) and its usage/tutorial/demo (another 10 slides). He will also provide overview of general SRM. If Ian wants more talks we can write presentation for dcache overview, vdt support for storage etc and ask Alain to present it.

Seventh agenda item

Tickets

Discussion

6256 IU - D0 transfers timing out. D0 has only one pool, which gets overloaded. Will move D0 use to reserved space, which has 44 pools.
