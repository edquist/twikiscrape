First agenda item

Status of storage tools

Discussion

Abhishek is ready to release a new version of tools. He is waiting for Brian to send him the latest fixes. The release will be ready within this month. Discussed the best way to release tools within VDT framework:

  1. For time being decided to release within vdt-dcache because of the following reasons:
    1. all current tools are for dcache, so it is logical to release them with dcache vdt cache
    2. the tool kit is packaged as rpms, so it will require min work to incorporate them into vdt-dcache installation procedure because it deals with rpms already
    3. most of tools may be used only by storage admin
  2. All new tools relevant to BeStMan should be packaged under common VDT infrastracture
  3. If we decide to move dcache to common VDT infrastructure we will move tools as well
