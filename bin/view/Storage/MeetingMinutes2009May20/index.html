First agenda item

Status of gPlazma problems at UChicago (Ted)

Discussion

UChicago has experienced intermittent gPlazma timeouts after upgrade to dcache 1.9.2-5. According to Rob gPlazma is still talking to an old GUMS, the load is low and usage pattern has not changed. The following steps have been done in order to mitigate the problem:
  1. saml-vo-mapping-cache-lifetime was set to 180 (from 0)
  2. it was attempted to get rid of gPlazma cell and configure it as a pluggable module within gridftp doors and srm. GridFtp? door started successfully, srm failed - Ted has committed a fix, it should be released soon.
  3. After these actions they still can see occasional timeouts, so it was advised to modify dcachesrm-gplazma.policy and allow the authorization via gridmapfile as well. The priority for grimapfile authorization is set to 1. It looks like the timeouts disappeared but still the cause is not clear. We will try to reproduce the problem on our test stand.

Second Agenda Item

Status of vdt-dcache configuration script, inclusion of toolkit (Neha)

Discussion

Neha is working on improving and modularizing code. She will include the installation and configuration of two rpms from the toolkit: dcache-core and Chronicle. The next release of vdt-dcache will also contain:

Third agenda item

Validation test suites update (Tanya)

Discussion

Tanya is working on consolidating all validation test suites on one web page. The results of the tests could be https://gw014k0.fnal.gov:8443/validation_tests/ - certificate required. The test include:

  1. dcache testing
    1. srm-fermi client
    2. replica manger tests
    3. disk utilization test
    4. space reservation tests
    5. information provider tests
    6. Flavia's S2 basic test
  2. BeStMan tests
    1. srm-lbnl, srm-fermi clients tests
    2. fullmode, gateway, gateway-Xrootd
    3. LBNL's srmtester

The page also contains ganglia monitoring for all test nodes (visible only on site), srmwatch, and cellinfo page. Tanya is planning to add xrootd monitoring as soon as she figures out where to get it and how to install it.
