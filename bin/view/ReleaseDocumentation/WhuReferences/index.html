Topic for INCLUDE-ing to create a "where used" in a BASETOPIC

on We should deprecate use of this topic for the %WHU% variable.

This topic is intended to be used as an include in all documents. It will provide some cross-references for a twikit topic:

  1. One showing all twiki topics where this topic is referenced with an INCLUDE
  2. Then a link which will find all "links" to this topic in the Integration documentation
  3. Finally, a link which will find all "links" to this topic in ALL documentation

The syntax for using this on a page is: 

   %INCLUDE{"Integration.WhuReferences"}%
or, easier on you: 
   %WHU%
It should almost always be placed AFTER the %STOPINCLUDE% in the topic. This will allow it to be visible when the topic is accessed directly, but it will not appear when a topic is INCLUDE'd in another topic.
