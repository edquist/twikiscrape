Use of trouble tickets in OSG

The OSG uses the GOC for central ticketing. From here there are several directions a ticket can be routed, to the VO, service, peering grid, or other OSG collaborator. If you would like to submit a ticket directly to the GOC click here. Automated ticket exchange from the GOC has been set up with the VDT, EGEE, ATLAS, CMS, and FNAL. If you would like to submit directly to a collaborator some ticketing links are located below.

Where to submit tickets

  • If you have a OSG Installation specific question please submit a ticket at Trouble Ticket.
  • You can also submit mail to goc@opensciencegrid.org, please directly request a ticket be created for prompt action.
  • For VDT support, email vdt-support@opensciencegrid.org this generates support tickets which we will answer on a best-effort basis.
  • Globus bugzilla is located here.
  • Troubleshooting users campaign is located here.
  • Troubleshooting http://software.grid.iu.edu/osg-1.2 1.2.32 campaign is located here.
