IPv6 Testing Log: Bestman2

This testing log is part of OSG IPv6 tests. This page captures the specific IPv6 compatibility testing process used for one software component, Bestman2. The more general IPv6 template for testing is here.

Initial Setup

This test used two hosts:

cmssrm2.hep.wisc.edu ?? 3.2

Installing software

Next, OSG software was installed on each system. The UCSD host was the client and the FNAL host was the server. And VOMS software was installed following the instructions here VOMSInstallation
