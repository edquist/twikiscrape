IPv6 Testing Log: HTTPD

This testing log is part of OSG IPv6 tests. This page captures the specific IPv6 compatibility testing process used for one software component, HTTPD (Apache). The more general IPv6 template for testing is here.

Initial Setup

This test used two hosts:

Host OS OSG Release
cabinet-10-10-4.t2.ucsd.edu CentOS release 6.5 (Final) 3.2
ipv6vm006.fnal.gov Scientific Linux Fermi release 6.6(Ramsey) 3.2

Installing software

Next, OSG software was installed on each system. The UCSD host was the client and the FNAL host was the server.

Host Package Version
cabinet-10-10-4.t2.ucsd.edu wget GNU Wget 1.11.4 Red Hat modified
ipv6vm006.fnal.gov httpd 2.2.15-39.sl6
