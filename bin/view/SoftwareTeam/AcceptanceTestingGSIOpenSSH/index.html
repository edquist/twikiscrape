How to test a fresh install

  1. Spin up two VM's and set up the EPEL/OSG repos on both of them.
  2. Choose one of the VM's, it will be the server VM. Consult these instructions to set up the server.
  3. From the other VM (client):
    • Install the necessary packages: 
      [root@client ~]$ yum install globus-proxy-utils gsi-openssh-clients
    • Initialize your proxy. After this, none of the gsi commands should prompt you for your password.
    • Connect to the server: 
      [user@client ~]$ gsissh -p 2222 <server hostname>
    • Copy a test file to the server: 
      [user@client ~]$ gsiscp -p 2222 testfile <server hostname>:/tmp
    • Connect to the server via SFTP and grab files: 
      [user@client ~]$ gsisftp -P 2222 <server hostname>
cd /tmp
get testfile
