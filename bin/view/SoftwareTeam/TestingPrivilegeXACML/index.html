4 Testing Instructions

4.1 In the gums server

Perform the following actions on the gums web interface, your DN must have admin privileges. See GUMS documentation for more information:

4.1.1 Add an account mapper with the following parameters:

name type persistence factory
test manual mysql

4.1.2 Add a user group name

group name type GUMS Access
test manual Read Self

4.1.3 Add a group to account mapping called

name user group account mapper
test test test

4.1.4 Add a host to group mapping

Hosts Group to Account Mapping
*/?*.fnal.gov test

4.1.5 In manual account mapping click add, and add the following:

User Group DN FQAN
test yourDN *

4.1.6 Install privilege-xacml 

[root@client ~]$ yum install privilege-xacml

4.1.7 Replace the privilege xacml

Replace /var/lib/tomcat[5|6]/webapps/gums/WEB-INF/lib/privilege-xacml-X.X.X.jar with the version you want to test: 

[root@client ~]$cp /usr/share/java/privilege-xacml.jar  /var/lib/tomcat[5|6]/webapps/gums/WEB-INF/lib/privilege-xacml-X.X.X.jar

4.2 In the Glexec node installation

4.2.1 Configure your glexec node

Configure the node so it points to your gums installation following the Glexec Config instrucctions

4.2.2 Add user test to the glexec node 

[root@client ~]$useradd test

4.2.3 Test glexc 

[user@client ~]$ voms-proxy-init
[user@client ~]$ export GLEXEC_CLIENT_CERT=/tmp/x509up_uNNN
[user@client ~]$ /usr/sbin/glexec /usr/bin/id
uid=13160(fnalgrid) gid=9767(fnalgrid) groups=65000(glexec00)
