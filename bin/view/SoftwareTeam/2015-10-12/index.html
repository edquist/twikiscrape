OSG Software Team

Discussions

GRAM/PBS. Last week, we started getting involved with an issue at University of Washington Seattle and their Hyak cluster. It seems that GRAM stops reporting on status of PBS jobs, causing the factory to stop sending new pilots. Jeff Dost was very involved in this ticket earlier, and got Hyak to switch to HTCondor-CE, which seems to be working great now. Jeff clarified that he can see this issue in factory monitoring very clearly, and it seems to go back as far as there are records, so it is unlikely to be (completely) due to any recent GRAM changes. The question has come up whether we can automatically detect and report on this issue; Jeff believes that he can describe the signature of the issue well enough that we could create a tool to watch for it. Also, Jeff will look through SW-1929 to see if there are any similarities. But given the scant support for GRAM at this time, it may be best to monitor for the issue and ask sites to move to HTCondor-CE.

Documentation. There is a new OSG documentation project starting, run by the GOC (Elizabeth and Kyle). They seem to be moving in a good direction, wanting to separate internal-only from external-facing documentation, remove old and unused documents, and have a clear, searchable site that is based on user needs. TimC will be involved in planning and direction, so you can bring questions and concerns to him. The timescale is fairly broad, so no rush.
