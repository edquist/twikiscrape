You can add sites by su'ing to the gridcat user on clu2 and running:

/opt/gridcat/sbin/gridcat-site.pl --action=add sitename

To modify an existing entry:

/opt/gridcat/sbin/gridcat-site.pl --action=modify sitename

To display all entry attributes:

/opt/gridcat/sbin/gridcat-site.pl --action=display
