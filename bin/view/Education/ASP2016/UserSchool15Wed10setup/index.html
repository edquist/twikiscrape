Wednesday Exercise 1.0: Before Starting...

Get Organized

  1. Create a directory on the OSG submit node (osg-ss-glidein-submit.chtc.wisc.edu) for the next set of exercises. Choose a name that matches the theme of the morning.
  2. Within that directory, create three sub-directories, one for each exercise. Give the sub-directories descriptive names, based on the content of the exercises:
    • Exercise 1.1: compiling code statically, program in C
    • Exercise 1.2: using a pre-compiled executable, BLAST
    • Exercise 1.3: using a wrapper script, GROMACS
